FTris Arena

FTRIS is a single and multiplayer game developed in an AngularJS framework. The visual content of the application is fully rendered in HTML and styled using CSS/ SASS. Javascript and JQuery make up the core object-oriented engine for game and data manipulation. Firebase was used as both the database as well as the server that stored and hosted this game.
